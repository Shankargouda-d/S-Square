## Project Description
A smart Kannada word processing system that:  
- Detects the **gender** of a Kannada noun (masculine, feminine, or neuter).  
- Fetches dictionary meanings from a CSV file.  
- Automatically generates **8 vibhakti forms** for any valid noun.  
- Exports all analyzed words to **Google Sheets** for tracking and analysis.
