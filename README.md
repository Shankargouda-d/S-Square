### Project Description

A smart Kannada word processing system that detects gender, fetches dictionary meanings from CSV, and automatically generates 8 vibhakti forms. The app also exports all analyzed words to Google Sheets for tracking and analysis.
### how to run 
Clone the project
Run flutter pub get
Add your Google Service Account JSON in google_sheets_exporter.dart
Enable Google Sheets API & share your Sheet
Run the app using: flutter run -d chrome
### result summary
Achieved accurate gender detection using CSV dictionary
Generated 8 vibhakti forms automatically for any valid noun
Successfully connected Flutter app to Google Sheets
All processed words saved with timestamp for future analysis
